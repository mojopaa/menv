# menv
Mojo venv

### Installation and Usage

- `git clone git@github.com:mojopaa/menv.git`
- `cd menv`
- `pip install pdm`
- `pdm install -d`
- Open venv:
    - Windows: `.venv\Scripts\activate`
    - Linux: `. .venv/bin/activate`
- Run `menv -h`
- Run `menv .test`
